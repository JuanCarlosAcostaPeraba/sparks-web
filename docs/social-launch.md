## LinkedIn

### Launch Post

I just released v0.1.0 of `sparks`, a small native CLI for capturing ideas, tasks, and nested thoughts without leaving the terminal.

The goal is simple: reduce the friction between having a thought and saving it somewhere useful.

What it does today:

- Add ideas and tasks quickly from the command line
- Create sub-ideas with `--parent`
- View nested thoughts as a tree
- Mark items as done or important
- Search by title
- Store everything locally in SQLite
- Export command output as JSON for scripting

Example:

```bash
sparks add "Ship the CLI"
sparks add --parent 1 "Write docs"
sparks tree
```

The project is early, and I am looking for feedback from developers, makers, and terminal-first users.
